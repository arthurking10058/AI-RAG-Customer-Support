# 项目讲解稿

## 30 秒版本

这是一个面向扫地机器人场景的智能客服项目，核心是 `Agent + 混合检索 RAG + Streamlit`。它不仅能回答产品知识问题，还能根据用户 ID 和月份生成结构化月度报告。

## 2 分钟版本

项目里，`Agent` 负责判断是否调用工具，`RAG` 负责补充知识，`middleware` 负责在普通咨询和报告模式之间切换提示词，`Streamlit` 负责稳定展示整个过程。

知识问答场景下，系统会先做 `Chroma + BM25 + RRF` 的混合检索，再把参考资料交给模型总结，并返回来源。报告场景下，系统会读取外部使用记录，输出固定的 `Markdown` 报告结构，便于截图和讲解。

这个项目的重点不是做企业级大系统，而是把一个个人项目收敛成可以演示、可以讲清、可以写进简历的稳定版本。

## 面试回答关键词
